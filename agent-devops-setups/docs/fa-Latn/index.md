# agent-devops-setups

Agent devops setups and policy documentation (Farsi Latin script)

In yek site-e mostanadat baraye tanzimat va siasat-haye DevOps amal ast. Shamel-e tarahi-e me'mari, barname-rizi-e ejraei, naqshe-bardari-e mahdude va savabet-e jalasat-e mortabet mi-bashad.

## Motevaye asli

- **Me'mari**: Tarahi-e me'mari-e system va tozihat-e ajza
- **Barname**: Barname-rizi-e projeh va naqshe rah-e ejraei  
- **Naqshe-bardari-e mahdude**: Ta'rif-e mahdude-ye karkardi va marz-ha
- **Savabet-e jalasat**: Bahs-ha va tasmim-giri-haye farayand-e tose'e

## Shorou'e sari'

Baraye daryaft-e jadid-tarin etela'at, mostanadat-e asli-e English ra moshahede konid, motevaye bomi-sazi be tadrij dar hale takmil ast.
