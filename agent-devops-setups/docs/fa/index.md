# agent-devops-setups

مستندات تنظیمات و سیاست‌های DevOps عامل

این یک سایت مستندات برای تنظیمات و سیاست‌های DevOps عامل است. شامل طراحی معماری، برنامه‌ریزی اجرایی، نقشه‌برداری محدوده و سوابق جلسات مرتبط می‌باشد.

## محتوای اصلی

- **معماری**: طراحی معماری سیستم و توضیحات اجزاء
- **برنامه**: برنامه‌ریزی پروژه و نقشه راه اجرایی  
- **نقشه‌برداری محدوده**: تعریف محدوده عملکردی و مرزها
- **سوابق جلسات**: بحث‌ها و تصمیم‌گیری‌های فرآیند توسعه

## شروع سریع

برای دریافت جدیدترین اطلاعات، مستندات اصلی انگلیسی را مشاهده کنید، محتوای بومی‌سازی به تدریج در حال تکمیل است.
